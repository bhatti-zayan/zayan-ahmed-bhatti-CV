# Zayan Ahmed Bhatti CV Website

## Project Description
This is my personal CV website built using HTML and CSS.
It showcases my profile, education, skills, experience, and contact information.
The website is responsive, has dark/light mode, downloadable CV, and simple animations.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Media Queries)

## Features
- Header with photo, name, title
- About Me section
- Education section
- Skills section
- Experience / Projects section
- Contact Information section
- Footer
- Dark/Light mode (CSS only)
- Downloadable CV
- Simple fade-in animations
- Responsive layout
